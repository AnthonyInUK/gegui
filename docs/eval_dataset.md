# 电商广告法多模态评测集

## 数据规模

- 样本总数：80
- 违规样本：60
- 正常样本：20
- 生成方式：离线合成电商海报图，违禁信息写入图片或与文案组合形成风险点
- 样本清单：`tests/ecommerce_ad/cases.json`
- 图片目录：`tests/ecommerce_ad/adversarial_cases/`

## 覆盖类型

| 类型 | 数量 | 目的 |
|---|---:|---|
| 图内藏字 | 4 | 验证纯文本规则无法看到的图片文字违规 |
| 拆字插符 | 5 | 验证符号插入、拆字规避的鲁棒性 |
| 繁体异体 | 4 | 验证繁体/异体字规避识别 |
| 谐音替换 | 4 | 验证拼音、谐音、混写表达识别 |
| 语义擦边 | 4 | 验证非黑名单字面命中时的语义判断 |
| 符号规避 | 3 | 验证竖线、间隔点、空格等规避表达 |
| 医疗用语 | 8 | 验证普通商品宣称治疗、消炎、降糖等风险 |
| 夸大功效 | 4 | 验证 100% 有效、一次见效、包治等夸大承诺 |
| 类目资质 | 8 | 验证普通食品/化妆品/家电类目承载保健、特妆、医疗器械功效 |
| 跨模态组合 | 10 | 验证图文组合后的暴富暗示、使用前后对比等隐性违规 |
| 站外导流 | 6 | 验证扫码、加 V、站外下单等平台治理风险 |
| 正常素材 | 8 | 验证普通商品描述不被误杀 |
| 正常促销 | 4 | 验证限时折扣、组合装等合规促销不被误杀 |
| 正常擦边词 | 4 | 验证热销、人气、高分好评等易误杀词 |
| 正常招聘 | 2 | 验证正常招募文案与暴富暗示的边界 |
| 正常健康描述 | 2 | 验证低糖、保湿等正常描述不被误杀 |

## 使用方式

真实调用模型评测：

```bash
.venv/bin/python src/eval_runner.py
```

只跑前 N 条做 smoke：

```bash
.venv/bin/python src/eval_runner.py --limit 5
```

从已有预测复算报告：

```bash
.venv/bin/python src/eval_runner.py --from-predictions docs/eval_predictions.json
```
